# Participant ids in the mock database

Participant ids in the mock database

## Usage

``` r
mock_person_ids(con)
```

## Arguments

- con:

  A connection from
  [`mock_aou_connect()`](https://louisahsmith.github.io/mockallofus/reference/mock_aou_connect.md).

## Value

An integer vector of `person_id`s.
