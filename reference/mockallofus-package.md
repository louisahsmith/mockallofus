# mockallofus: A Local Mock Database for Developing All of Us Analyses

Builds a local DuckDB database with fully synthetic data that mirrors
the table and column structure of the All of Us Research Program Curated
Data Repository (OMOP CDM plus All of Us extensions). Lets users develop
and test 'allofus' package analysis code locally, without the paid
Researcher Workbench. The synthetic data is randomly generated and does
not reflect real All of Us data. The same analysis code is intended to
run against the mock database locally and against the real data on the
Workbench, with a small set of clearly-documented exceptions.
